# Ритейл.Щит Актив

Прототип автоматизированной оценки киберрисков и расчёта страховой премии
для розничной торговли, e-commerce и маркетплейсов.

Разработан в рамках студенческого конкурса по кибербезопасности и киберстрахованию
«Киберстипендия» командой **КиберКаска** (Финансовый университет, 2026).

## Что это

Веб-приложение реализует методологию из конкурсной работы:

- скоринг по анкете А-Е с учётом K_неопр и правил согласованности (А.26);
- проверка семи стоп-факторов из обновлённой главы 12 Блока 1;
- классификация по пяти классам риска (Low / Medium / High / Very High / Critical);
- FAIR-оценка ALE с учётом каскадного эффекта (CascadeFactor);
- расчёт убытка от перерыва деятельности (BI) по формуле Блока 3, п. 3.1;
- расчёт страховой премии по полной формуле:
  `P = S × T × K_отрасли × K_скора × K_размера × K_истории × K_сервиса × K_внеш × K_аудита`;
- сценарный анализ по шести подмоделям (ransomware, утечка данных, BI, контрактор, BEC, простой);
- индекс киберстрахуемости (CII) и его четыре компонента;
- индекс эволюции риска (ΔScore) для динамического ценообразования при продлении;
- Loss Ratio для оценки портфельной устойчивости;
- HTML-дашборд карты отраслевых рисков;
- скачиваемый HTML-отчёт со списком вопросов и ответов клиента.

## Технологический стек

- Java 21
- Spring Boot 3.5
- Thymeleaf + Layout Dialect
- Jackson (загрузка `retail_risk_map.json`)
- Maven

## Локальный запуск

```bash
mvn spring-boot:run
```

Затем открыть в браузере: <http://localhost:8080>

## Сценарий работы

1. Лендинг → выбор пакета (Лайт / Базовый / Стандарт / Премиум).
2. Заполнение данных компании.
3. Чек-лист предварительного отбора (7 жёстких стоп-факторов).
4. Анкета по разделам А-Е (А-Д универсальные, Е — отраслевой для ритейла).
5. Экран результатов: класс риска, скоры, CII, FAIR + CascadeFactor, BI, премия,
   Loss Ratio, сценарный анализ.
6. HTML-отчёт + HTML-дашборд карты рисков.
